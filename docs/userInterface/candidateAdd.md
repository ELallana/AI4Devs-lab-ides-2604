It is form page for filling with the data from the cadadate. 

It must have text-boxes for all the text properties in the `candidate` entity. They must be writable.

For the `cv` property of `candidate` there is a widget to select a file from the local file system to upload.

Also has two bottons:

- `cancel` botton. With the text "Cancel".
- `submit` botton. With the text "Submit".

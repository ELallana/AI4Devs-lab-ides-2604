id: N4
name: The candidate data can not be inserted into the database.
---
## Scope
Backend.

## Related use cases
P3, P4.

## Prerequisites

## Initiator
Arrives to the `backendService` a request of a candidate.

## Steps
1. The data validation fails.
2. The `candidate` data is sent to the DB client to insert in the DB.
3. The DB client returns an error.
4. Return an error to the REST client.

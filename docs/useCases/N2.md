## Related uses cases
P3.

## Prerequisites
The user is in the `candidateAdd` page.

## Initialicer
The user clicks the button `submit`.

## Steps
1. The data in the page for the `candidate` entity is validated.
2. The `candidate` entity is send to the backend.  
3. The backend responses with an error.
4. Show a message in the page in red text indicating that is not posible to save the candidate and wich is the error.
